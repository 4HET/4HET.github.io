c++string

string与c语言char*区别

char是一个指针，而string是一个类，里面封装了char。并且string类内部封装了很多成员方法，例如查找find，拷贝copy，删除delete，替换replace，插入insert。string不用担心复制越界和取值越界。

构造

灵活使用即可。

赋值

有两种方式，一种是等号赋值，另一种是assign赋值

字符串拼接

有两种方式，第一种直接使用+=，第二种使用成员函数append

字符串查找与替换

查找：查找指定字符是否存在
替换：在指定位置替换字符串

string字符串比较

比较方式是按字符的ASCII码进行对比，=返回 0，>返回 1，<返回 -1。

string字符存取

string中单个字符存取方式有两种，第一种是通过[]方式取字符，第二种是通过at方法获取字符。

string插入和删除

插入和删除的起始坐标都是从0开始。

string子串

功能：从字符串中获得想要的子串。
